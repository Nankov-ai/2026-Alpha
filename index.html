Google Drive
Liga o Google Drive para importar ficheiros directamente para o chat e para o analisador de projectos.

Como configurar:
1. Vai a console.cloud.google.com → New Project
2. Activa Google Drive API e Google Picker API
3. Cria credenciais OAuth 2.0 (tipo: Web application, authorized origin: o teu URL GitHub Pages)
4. Cria também uma API Key (restrita ao teu domínio)
5. Cola os valores aqui
Sincronização entre dispositivos
Usa a API do GitHub Gist para guardar os dados na web. Assim podes aceder do telemóvel, computador ou qualquer browser.

Como configurar:
1. Vai a github.com → Settings → Developer settings → Personal access tokens → Fine-grained
2. Cria token com permissão Gists: Read and Write
3. Cola o token aqui e clica em "Criar Gist"
